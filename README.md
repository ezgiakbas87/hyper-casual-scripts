# rolling-the-ball-game-script

I made a Hyper Casual Ball Game. Here you can find all the scripts I wrote and some in-game screenshots.
The game has 5 levels, 1 menu and 1 exit screen.
In-game purchases:
- 3 coin packs
- 4 particles
